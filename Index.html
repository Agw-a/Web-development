What is CSS?

CSS ( Cascading Style Sheets ) is a style sheet language used to style HTML elements. It describes the presantation of web pages including colors, layout and fonts. For example the CSS code below styles paragraphs to color red while the body's background color is white. 

            
              p { color: red; }
              body { background-color: white; }

Anatomy of CSS ruleset

This section disects the anatomy of CSS. Considering the CSS code above, a CSS ruleset comprises a selector, declaration, propoerties and property value as in the image below.

Selector

"p" is a selector in CSS. The selector is the name at the start of a rule-set defining the HTML element that is to be styled .

Declaration

A declaration is a single rule that defines the properties you want to style e.g "color red".

Property

The desired way of styling a HTML Document e.g "color" is a property of "p".

Property value

This defines the specific way of defining the property e.g "red" is the value of the property "color".


NOTE:
  • Each rule-set must be wrapped in curly braces {}
  • With each declaration, use a colon (:) to separate the property from the values.
  • Use a semi-colon (;) to separate each declaration from the nest one i.e. 
                        
                        p {
                            color: red;
                            width: 500px;
                            border: 1px solid black;
                            }
  • You can select multiple HTML elements to be defined/styled and separate each of them by a comma e.g 
                        
                        p, li, h1 {
                            color: red;
                          }

CSS selectors

CSS selectors find the specific HTML element to be syled.These selectors include the following;

  • Element selector
  • ID selector
  • Class selector
  • Pseodo selector
  • Universal selector
  • Grouping selector
Selector Purpose Example
Element Selector All HTML of the specific tag or the element name. Here, all "p" elements on the page will be center-aligned, with a red text color: 
p { text-align: center; color: red; }
ID Selectors The elements on a page with a specified ID. Each HTML element has a unique ID that can be traced in CSS. To select an element with a specific id, write a hash (#) character, followed by the id of the element. The CSS rule below would be applied to HTML elements of "id=indent-line"
#indent-line { text-align: center; color: red;
Class Selector Elements of a HTML page with the same class/ specific class attribute. To select elements with a specific class, write a period (.) character, followed by the class name. HTML element class="my-class" would be selected with the CSS selector below;
.my-class { text-align: center; color: red;
Pseudo selector For the specified elements but only when in a given state. e.g only when cursor hovers over the text. a:hover
selects "a" but only when the mouse pointer is hovering over the link
Universal selector Selects all the HTML elements in the page *{ text-align: center; color: blue; }
Grouping selector Applied if HTML elements e.g h1, h2, and p have the same definitions e.g color red and centered text alignment, a grouping selector is applied to reduce the bulk of code. h1, h2, p { text-align: center; color: red;}
